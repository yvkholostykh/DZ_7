# DZ_7
Лабораторная работа №7
Проект по работе с API на Python
Этот проект демонстрирует работу с публичными API: получение постов из JSONPlaceholder и данных о погоде из OpenWeatherMap с визуализацией результатов.

Функциональность
Отправка GET‑запросов к API.

Обработка JSON‑ответов.

Передача параметров в запросах.

Анимация загрузки.

Сохранение данных в форматах JSON и CSV.

Создание графиков температуры и анализа данных.

Использование эмодзи для улучшения пользовательского опыта (UX).

Требования
Python 3.7 или выше.

Библиотеки (см. requirements.txt).

Установка зависимостей
Создайте файл requirements.txt со следующим содержимым:

txt
requests>=2.25.0
matplotlib>=3.4.0
Установите зависимости командой:

bash
pip install -r requirements.txt
Настройка API
Перейдите на .

Зарегистрируйтесь или войдите в аккаунт.

В личном кабинете откройте раздел API Keys.

Создайте новый API‑ключ или скопируйте существующий.

Подождите 5–10 минут для активации ключа.

Инструкция по запуску
Шаг 1. Подготовка файлов
Сохраните основной код в файл main.py.

Создайте файл requirements.txt с зависимостями (см. выше).

Шаг 2. Настройка API‑ключа
Откройте файл main.py в текстовом редакторе.

Найдите строку:

python
api_key = "YOUR_API_KEY_HERE"
Замените "YOUR_API_KEY_HERE" на ваш реальный API‑ключ от OpenWeatherMap.

Сохраните изменения.

Шаг 3. Запуск проекта
Откройте терминал/командную строку.

Перейдите в папку с проектом:

bash
cd C:\Users\Holos\Desktop\DZ_7_Holostyh_yura
(замените путь на актуальный для вашей системы)
3. Запустите скрипт:

bash
python main.py
Шаг 4. Взаимодействие с программой
Программа выведет приветственное сообщение и начнёт загрузку постов из JSONPlaceholder.

После отображения первых 5 постов программа запросит название города:

🗺️  Введите название города для проверки погоды:
Введите название города (например, Moscow, London, Aчинск) и нажмите Enter.

Программа получит данные о погоде и создаст отчёты.

Ожидаемые результаты
После успешного выполнения в папке data/ будут созданы:

posts_data.csv — первые 5 постов из JSONPlaceholder;

weather_report.json — отчёт о погоде для указанного города;

weather_plot.png — график температуры;

combined_analysis.png — комбинированный график с анализом данных.

Структура вывода программы
Приветственное сообщение.

Загрузка и отображение первых 5 постов.

Сохранение постов в CSV.

Запрос города для проверки погоды.

Получение и отображение данных о погоде.

Сохранение отчёта о погоде в JSON.

Создание и сохранение графиков.

Финальное сообщение об успешном завершении.

Возможные ошибки и их решение
Ошибка	Причина	Решение
401 Unauthorized	Неверный или неактивированный API‑ключ	Проверьте ключ, дождитесь активации (5–10 мин)
404 Not Found	Город не найден	Проверьте написание названия города
NameError: 'first_five' not defined	Устаревший код	Используйте актуальную версию main.py из этого README
Проблемы с зависимостями	Не установлены библиотеки	Выполните pip install -r requirements.txt
